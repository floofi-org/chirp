Floofi Voice Generator has officially shut down

    The Floofi Voice Generator service shut down on April 17th, 2025 and you are not able to generate new content.

    If you would like to build an application using Floofi Voice Generator's models, which were previously exposed via the API, feel free to contact us so we can discuss a partnership.

    We are sorry to abandon the Floofi Voice Generator project and we are looking forward to building better and more useful applications for everyone.
